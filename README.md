# MERN-Book-Finder

This application lets a user to search for new books to read. They can save or delete the books on their account which they would like to purchased in future.

## Technologies used:

- MongoDB
- Express.js
- React.js
- Node.js
- JavaScript
- GraphQL API
- Apollo Server
- Heroku

## Installation

- npm install
